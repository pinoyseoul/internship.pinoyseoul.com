## About

This is the official landing page for the PinoySeoul Digital Media Internship Program. The site is designed to attract potential interns and university partners, providing all the information needed to understand and apply for the program.

**PinoySeoul Media Enterprise** is a digital media company bridging Korean and Filipino creative communities through broadcasting, publishing, and media technology. The internship program offers structured OJT across four tracks: Broadcasting & Production, Digital Marketing & Publishing, Business Development & Outreach, and Media Infrastructure & Tech.

## Features

- 🎨 Modern, Gen Z-friendly dark-mode design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🔍 SEO-optimized with structured data (JSON-LD), Open Graph, and Twitter Cards
- ⚡ Lightweight static site — no frameworks, no build step
- 🗺️ Auto-generated `sitemap.xml` via GitHub Actions
- ♿ Accessible with ARIA attributes and keyboard navigation

## Tech Stack

- HTML5 + Semantic Markup
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Intersection Observer API, scroll effects)
- GitHub Pages (hosting)
- GitHub Actions (sitemap generation)
